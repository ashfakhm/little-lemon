# 🍋 Little Lemon Restaurant

A premium, high-fidelity table reservation platform for the Little Lemon restaurant, featuring a modern Mediterranean design and a robust booking system.

> [!IMPORTANT]
> This is the project I completed as part of the Meta Front-End Developer course to earn my certificate.

## ✨ Features

- **Table Reservation System**: A fully functional booking flow that manages date, time, number of guests, and seating preferences.
- **Dynamic Availability**: Integrates with a simulated API to fetch available time slots for any given date.
- **Form Integrity**: Robust client-side validation using **Formik** and **Yup**, ensuring error-free submissions.
- **Responsive & Accessible**: Mobile-first architecture built with **Tailwind CSS**, adhering to ARIA standards for accessibility.
- **Premium Aesthetics**: Custom branding with **Markazi Text** and **Karla** typography, capturing the heritage of Little Lemon.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: React Hooks (`useReducer`, `useState`)
- **Forms**: [Formik](https://formik.org/) & [Yup](https://github.com/jquense/yup)
- **Routing**: [React Router v6](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Testing**: [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/)

## 🚀 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/your-username/little-lemon.git
cd little-lemon
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm start
```

### 4. Run tests
```bash
npm test
```

The visual language of this project is documented in [designguideline.md](designguideline.md). It defines:
- **Colors**: Lemon Yellow (#F4CE14) and Forest Green (#495E57).
- **Typography**: Markazi Text for headings, Karla for body copy.
